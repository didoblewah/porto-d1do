Arcade Game: Hoops Monster

Dalam proyek ini, saya berperan sebagai software developer dalam tim yang membuat arcade game "Hoops Monster". Game ini mengharuskan pemain memasukkan bola ke dalam mulut monster yang dilengkapi dengan sensor infrared (IR) untuk mendeteksi bola yang masuk. Sistem ini terintegrasi dengan Unity Engine untuk menampilkan skor secara real-time. Kami menggunakan Arduino Uno sebagai mikrokontroler utama yang diprogram melalui Arduino IDE untuk memproses input dari sensor dan mengirimkan data ke Unity melalui komunikasi serial. Selain itu, saya juga menggunakan Canva untuk merancang elemen visual dan UI yang menarik untuk meningkatkan pengalaman bermain.

Arduino Uno Arduino IDE Unity C# Canva
